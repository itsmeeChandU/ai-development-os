# Code Of Conduct

This project welcomes thoughtful, practical contributions.

Please:

- be respectful,
- assume good intent,
- prefer evidence over status,
- keep criticism specific and actionable,
- avoid sharing private data in public examples,
- help make complex systems easier for others to understand.

Maintainers may remove comments, issues, or contributions that are abusive,
harassing, spammy, or unsafe for public review.
