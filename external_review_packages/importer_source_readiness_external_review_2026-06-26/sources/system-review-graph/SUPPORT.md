# Support

Use GitHub Issues for:

- bug reports,
- feature requests,
- example requests,
- documentation gaps,
- questions about manifest structure.

When asking for help, include:

- the command you ran,
- your Python version,
- the manifest section involved,
- the generated report section that looked wrong,
- a redacted or fictional manifest snippet.

Please do not paste private data, credentials, production records, or customer
information into public issues.
